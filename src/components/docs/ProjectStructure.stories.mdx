# Project structure
| Folder | Description |
| :-: | :-- |
| dist | The dist folder contains dynamically generated when using the npm run build command and includes the generated production ready HTML files and assets. |
| public | The public folder contains files that will be included in your final build. |
| assets | The assets directory contains your static files that can be used within your application. | 
| components | The components directory contains your Vue.js components. | 
| plugins | The plugins directory contains the utilities to apply various functions in the application. |
| router | The router directory contains the router configuration. |
| store | The store directory contains your Vuex Store files. | 
| views | The pages directory contains your application views and routes. | 
| .env | The default configuration can be overwritten. [(Read more here.)](https://cli.vuejs.org/guide/mode-and-env.html) |
